# DAN

> **Cloud · DevOps · Full-Stack Developer**

서비스 설계부터 배포 자동화, 프론트엔드와 백엔드까지 전 구간을 아우르며 **Cloud Native · DevOps · Full-Stack** 세 축으로 커리어를 쌓아가는 개발자입니다.

온프레미스에서 클라우드로 넘어가는 전환기 한가운데에서, 지금은 **AI를 페어 프로그래머처럼** 활용해 레거시 애플리케이션과 인프라를 현대화하고 있습니다. 코드를 직접 찍어내는 대신, 문제를 정의하고 결과를 검증하는 역할에 집중합니다.

<br>

## 🧭 What I'm Building Toward

* **Cloud Native** — 컨테이너와 오케스트레이션(Kubernetes)을 기반으로, 트래픽에 유연하게 반응하고 스스로 회복하는 인프라를 지향합니다.
* **DevOps Culture** — CI/CD 파이프라인으로 빌드-테스트-배포를 자동화하고, 휴먼 에러를 줄여 개발자가 코드에만 집중할 수 있는 환경을 만듭니다.
* **Full-Stack Ownership** — 화면부터 쿼리까지, 기능이 사용자에게 닿는 전 과정을 설계하고 책임지며, 어느 구간이든 파고들어 개선할 수 있는 이해도를 지향합니다.
* **AI-Driven Modernization** — Kiro, Claude 같은 AI 에이전트를 개발 워크플로우에 녹여, 레거시 코드 분석부터 리팩터링, 테스트 작성, IaC 전환까지의 사이클을 가속합니다.
